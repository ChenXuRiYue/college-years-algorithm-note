```cpp
ll quickly_pow(ll x,ll n,ll mod)
{
    ll res=1;//用来返回结果。
    while(n>0)
    {
        if(n&1)res=res*x%mod;
        x=x*x%mod;
        n>>=1;
    }
    return res;
}
```

- 分析模板
  - 当前的剩余部分是否可以被$x^2$（$x$实际意义上是多少次方）整除.
    - 如果可以不需要处理，
    - 如果不可以，余出来的部分必然是x,将它先乘在结果之上即可。
