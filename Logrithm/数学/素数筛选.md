##### $欧拉筛法，线性筛法$

- 和埃氏筛法之间有什么差别？
  - 埃氏筛法去合数的过程中。可能会重复经理某一些数字，存在一些多余的计算。

```cpp
void init() {
  for (int i = 2; i < MAXN; ++i) {
    if (!vis[i]) 
      pri[cnt++] = i;
    for (int j = 0; j < cnt; ++j) {
      if (1ll * i * pri[j] >= MAXN) break;
      vis[i * pri[j]] = 1;
      if (i % pri[j] == 0) 
        break;
    }
  }
}
```



- 来自jly的板

```cpp
int mp[maxn+1];
vector<int>primes;

for (int i = 2; i <= maxn; i++)
{
    if (!mp[i])
    {
        mp[i] = i;
        primes.push_back(i);
    }
    for (auto p : primes)
    {
        if (p * i > maxn)
            break;
        mp[i * p] = p;
        if (i % p == 0)
            break;
    }
}
```

- 这个可以方便的拆分素数。
