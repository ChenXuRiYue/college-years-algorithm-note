#### 定义

对于线性同余方程$ax\equiv1(mod \quad b)$

称x为a mod b的逆元记作$a^{-1}$



#### 性质

逆元存在：

1. 当$gcd(a,mod)=1$时，逆元才有解。



#### 逆元计算

##### 费马小定理：）

1. 内容：

$$
1. 假如a是一个整数。m是质数。\\
a^m\equiv a(mod\quad m)\\
$$

$$
2. 假设m不是质数,gcd(a,m)=1\\
a^{m-1}\equiv 1(mod\quad m)\\
a*a^{m-2}\equiv 1(mod\equiv m)\\
综上a^{m-2}是a的逆元。
$$

- 前提约束：mod数是质数。

其实就是写一个快速幂运算。

```cpp
//inverse_element.
ll mod;
ll qpow( ll x , ll n , ll p = mod)
{
    ll res = 1;
    while (n > 0) {
        if (n & 1) res = res * x % p;
        x = x * x % mod;
        n >>= 1;
    }
    return res;
}

ll inv(ll x , ll p = mod)
{
    return qpow( x , p - 2);
}
/*
* 1. mod定义
* 2. 使用前提：p是质数，且x，p互质。
*/
```
